## 화면 구성
- [메인 페이지][]  
- 랜딩 페이지  
- 로그인, 로그아웃
- 회원 가입, 아이디 찾기, 비밀번호 찾기  
- 프로필 선택, 프로필 편집  
- 검색 페이지

### 화면 플로우  
1. 메인 페이지 => 프로필 선택 => 로그아웃 => 랜딩 페이지 => '새로워진 타잉을 만나보세요' 클릭 => 로그인 페이지 (=> 아이디 찾기, 비밀번호 찾기, 회원 가입)  
2. 프로필 선택 => 프로필 전환 => 프로필 선택 => 프로필 편집  
3. 메뉴 => 검색 아이콘 => 검색 페이지
